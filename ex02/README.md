## 超高機能電卓（ex02/calc.py）
### 追加機能
- 四則演算：＋以外の四則演算をする. フロントエンドは"×","÷"で表示されているが計算前に"*","/"に変える
- クリア(C)：entryに入力されている数字，数式の1文字をdeleteする
- オールクリア(AC)：entryに入力されている数字，数式の文字列全体をdeleteする
- 一般に使かわれている電卓と同じ配置
